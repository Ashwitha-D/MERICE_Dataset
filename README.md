# MERICE_Dataset
Multimodal Classroom Engagement Dataset

The MERICE (Multimodal Engagement Recognition in Classroom Environments) Dataset is a multimodal dataset designed for analysing classroom engagement. It consists of 1,188 video clips, each with a duration of 10 seconds, capturing diverse classroom scenarios. It is intended to support research in areas such as multimodal learning, affective computing, and engagement recognition. 
Dataset Contents:

Videos: 1,188 short clips (10 seconds each)

Annotations: Engagement labels provided in a CSV file
Each video is annotated with one of the following engagement levels:
* High
* Medium
* Low
